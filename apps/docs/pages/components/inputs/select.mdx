# Select Component Documentation

The `Select` component provides a powerful and flexible dropdown selection interface for React Native applications. It supports grouped options, custom rendering, and search functionality, making it a highly customizable tool for building intuitive UIs.

---

## Props

The `Select` component accepts the following props:

| Prop                   | Type                                                                                                           | Description                                                                                              |
| ---------------------- | -------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------- |
| `defaultValue`         | `string`                                                                                                       | The initial value of the `Select` component.                                                             |
| `value`                | `string`                                                                                                       | The controlled value of the `Select` component.                                                          |
| `onChange`             | `(value: string) => void`                                                                                      | Callback function triggered when the value changes.                                                      |
| `contentClassName`     | `string`                                                                                                       | Additional class names for the dropdown content.                                                         |
| `triggerClassName`     | `string`                                                                                                       | Additional class names for the trigger element.                                                          |
| `valueClassName`       | `string`                                                                                                       | Additional class names for the displayed selected value.                                                 |
| `placeholder`          | `string`                                                                                                       | Placeholder text when no option is selected.                                                             |
| `options`              | `string[]`                                                                                                     | The options to display in the dropdown. Supports Array of strings                                        |
| `options`              | `{ label:string, value:string }[]`                                                                             | The options to display in the dropdown. Supports flat options.                                           |
| `options`              | `{ group:string, items: Option[] }`                                                                            | The options to display in the dropdown. Supports grouped options here Option is previous two types union |
| `renderOption`         | `(data: { option: { label:string, value:string}, isSelected: boolean, setOption: () => void }) => JSX.Element` | Custom renderer for options.                                                                             |
| `renderSelectedOption` | `(data: { option: SelectItemOption }) => JSX.Element`                                                          | Custom renderer for the selected option.                                                                 |
| `searchEnabled`        | `boolean`                                                                                                      | Enables or disables the search input in the dropdown.                                                    |
| `label`                | `string`                                                                                                       | Label for the `Select` component.                                                                        |
| `helperText`           | `string`                                                                                                       | Helper text displayed below the `Select` component.                                                      |
| `message`              | `string`                                                                                                       | Feedback message displayed below the component (e.g., success or error messages).                        |
| `error`                | `boolean`                                                                                                      | Indicates an error state.                                                                                |
| `success`              | `boolean`                                                                                                      | Indicates a success state.                                                                               |

## Usage

### Basic Example

```tsx
import { Select } from '@nativebricks/core';
import React, { useState } from 'react';
import { View } from 'react-native';

export default function SelectPage() {
  const [value, setValue] = useState('');

  return (
    <View className="container">
      <Select
        options={['Option 1', 'Option 2', 'Option 3']}
        placeholder="Select an option"
        value={value}
        onChange={setValue}
        label="Simple Select"
        helperText="Choose one of the options."
      />
    </View>
  );
}
```

### Grouped Options

```tsx
<Select
  options={[
    {
      group: 'Group 1',
      items: ['Option 1', 'Option 2', 'Option 3'],
    },
    'Option 4',
    'Option 5',
  ]}
  placeholder="Select an option"
  value={value}
  onChange={setValue}
  label="Grouped Options"
/>
```

### Custom Option Rendering

```tsx
<Select
  options={['Option A', 'Option B', 'Option C']}
  renderOption={(option) => (
    <Pressable
      className="p-2 bg-primary/25 flex-row items-center gap-2 rounded-xl hover:bg-primary/15"
      onPress={option.setOption}
    >
      {option.isSelected && <CheckIcon className="text-primary" />}
      <Text>{option.option}</Text>
    </Pressable>
  )}
  placeholder="Select an option"
  value={value}
  onChange={setValue}
  label="Custom Options"
/>
```

## Features

- **Grouped Options**: Organize options into logical groups for easier navigation.
- **Custom Rendering**: Fully customize how options and the selected value are displayed.
- **Search Functionality**: Quickly filter options using a built-in search bar.
- **Accessibility**: Includes ARIA labels and error messages for enhanced accessibility.

---

## Integration Notes

1. Ensure to pass valid options. Options can be strings or objects with `label` and `value` properties.
2. Use the `onChange` prop to handle value updates effectively.

---

The `Select` component is designed to provide a rich and seamless dropdown experience in React Native applications.
