# MultiSelect

A select component that allows users to select multiple options from a dropdown list.

## Installation

```bash
npm install @ui/components
```

## Import

```tsx
import { MultiSelect } from '@ui/components';
```

## Usage

```tsx
export default function MultiSelectDemo() {
  const [values, setValues] = useState<string[]>([]);

  return (
    <MultiSelect
      value={values}
      onChange={setValues}
      placeholder="Select options"
      options={[
        { value: 'react', label: 'React' },
        { value: 'vue', label: 'Vue' },
        { value: 'svelte', label: 'Svelte' },
      ]}
    />
  );
}
```

## Features

- Multiple option selection
- Search functionality
- Customizable chip display
- Group options support
- Custom option rendering
- Form integration
- Keyboard navigation

## Props

### Root

| Prop                   | Type                                                  | Description                      |
| ---------------------- | ----------------------------------------------------- | -------------------------------- |
| `value`                | `string[]`                                            | Selected values                  |
| `onChange`             | `(value: string[]) => void`                           | Selection change callback        |
| `options`              | `SelectOptions<Option>`                               | Array of options to display      |
| `placeholder`          | `string`                                              | Placeholder text                 |
| `searchEnabled`        | `boolean`                                             | Enable search functionality      |
| `contentClassName`     | `string`                                              | Class name for content container |
| `triggerClassName`     | `string`                                              | Class name for trigger button    |
| `valueClassName`       | `string`                                              | Class name for selected values   |
| `renderOption`         | `(data: OptionRenderData) => JSX.Element`             | Custom option renderer           |
| `renderSelectedOption` | `(data: { option: SelectItemOption }) => JSX.Element` | Custom selected value renderer   |

### With Search

