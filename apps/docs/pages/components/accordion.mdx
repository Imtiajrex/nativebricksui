# Accordion

A vertically stacked set of interactive headings that each reveal a section of content.

## Installation

```bash
npm install @ui/components
```

## Import

```tsx
import { Accordion } from '@ui/components';
```

## Usage

```tsx
export default function AccordionDemo() {
  return (
    <Accordion
      data={[
        {
          value: 'item-1',
          title: 'Is it accessible?',
          content: 'Yes. It adheres to the WAI-ARIA design pattern.',
        },
        {
          value: 'item-2',
          title: 'Is it styled?',
          content: 'Yes. It comes with default styles that match your theme.',
        },
      ]}
    />
  );
}
```

## Features

- Collapsible sections with animated transitions
- Keyboard navigation support
- Customizable styling for each section
- WAI-ARIA compliant
- Mobile responsive

## Props

### Root

| Prop                   | Type                  | Description                     |
| ---------------------- | --------------------- | ------------------------------- |
| `data`                 | `AccordionDataType[]` | Array of accordion items        |
| `contentClassName`     | `string`              | Class name for content sections |
| `contentTextClassName` | `string`              | Class name for content text     |
| `triggerClassName`     | `string`              | Class name for triggers         |
| `triggerTextClassName` | `string`              | Class name for trigger text     |
| `itemClassName`        | `string`              | Class name for individual items |

### AccordionDataType

```tsx
type AccordionDataType = {
  title: string;
  content: string;
  value: string;
  contentClassName?: string;
  contentTextClassName?: string;
  triggerClassName?: string;
  triggerTextClassName?: string;
  itemClassName?: string;
};
```

## Examples

### Basic Accordion

```tsx
<Accordion
  data={[
    {
      value: 'section-1',
      title: 'Section 1',
      content: 'Content for section 1',
    },
    {
      value: 'section-2',
      title: 'Section 2',
      content: 'Content for section 2',
    },
  ]}
/>
```

### Styled Accordion

```tsx
<Accordion
  triggerClassName="bg-muted hover:bg-muted/80"
  contentClassName="p-4"
  data={[
    {
      value: 'custom-1',
      title: 'Custom Section',
      content: 'Custom content with styling',
      triggerTextClassName: 'font-bold',
      contentTextClassName: 'text-muted-foreground',
    },
    {
      value: 'custom-2',
      title: 'Another Section',
      content: 'More custom content',
      itemClassName: 'border rounded-lg',
    },
  ]}
/>
```

### Individual Item Styling

```tsx
<Accordion
  data={[
    {
      value: 'item-1',
      title: 'Special Item',
      content: 'This item has custom styling',
      triggerClassName: 'bg-primary text-primary-foreground',
      contentClassName: 'bg-primary/10',
    },
    {
      value: 'item-2',
      title: 'Regular Item',
      content: 'This uses default styling',
    },
  ]}
/>
```
