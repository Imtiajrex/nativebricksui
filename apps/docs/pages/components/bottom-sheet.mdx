# BottomSheet

A panel that slides up from the bottom of the screen, providing additional content or actions while maintaining context.

## Installation

```bash
npm install @ui/components
```

## Import

```tsx
import { BottomSheet } from '@ui/components';
```

## Usage

```tsx
export default function BottomSheetDemo() {
  const [open, setOpen] = useState(false);

  return (
    <>
      <Button onClick={() => setOpen(true)}>Open Sheet</Button>
      <BottomSheet open={open} onOpenChange={setOpen} title="Sheet Title">
        <div className="p-4">
          <p>Sheet content goes here</p>
        </div>
      </BottomSheet>
    </>
  );
}
```

## Features

- Smooth slide-up animation
- Gesture-based interactions (swipe to dismiss)
- Snap points for different heights
- Backdrop overlay with tap to dismiss
- Customizable content area
- Mobile-first design

## Props

### Root

| Prop               | Type                      | Description                      |
| ------------------ | ------------------------- | -------------------------------- |
| `open`             | `boolean`                 | Whether the sheet is open        |
| `onOpenChange`     | `(open: boolean) => void` | Callback when open state changes |
| `title`            | `string`                  | Optional title for the sheet     |
| `description`      | `string`                  | Optional description text        |
| `snapPoints`       | `number[]`                | Array of snap points (0-1)       |
| `defaultSnapPoint` | `number`                  | Initial snap point index         |
| `children`         | `React.ReactNode`         | Content of the sheet             |
| `className`        | `string`                  | Optional class name for styling  |
| `overlayClassName` | `string`                  | Optional class name for overlay  |

## Examples

### Basic Sheet

```tsx
<BottomSheet open={open} onOpenChange={setOpen} title="Settings">
  <div className="p-4 space-y-4">
    <h3>Preferences</h3>
    <p>Configure your preferences here</p>
  </div>
</BottomSheet>
```

### With Snap Points

```tsx
<BottomSheet
  open={open}
  onOpenChange={setOpen}
  snapPoints={[0.25, 0.5, 0.9]}
  defaultSnapPoint={1}
  title="Media Gallery"
>
  <div className="p-4">
    <div className="grid grid-cols-3 gap-2">{/* Gallery items */}</div>
  </div>
</BottomSheet>
```

### Custom Styled Sheet

```tsx
<BottomSheet
  open={open}
  onOpenChange={setOpen}
  title="Cart"
  className="bg-background/95 backdrop-blur supports-[backdrop-filter]:bg-background/60"
  overlayClassName="bg-black/20"
>
  <div className="p-4">
    <div className="space-y-4">
      <h3 className="font-medium">Shopping Cart</h3>
      {/* Cart items */}
    </div>
  </div>
</BottomSheet>
```
