# Tabs

A tabbed interface component that allows users to switch between different views.

## Installation

```bash
npm install @ui/components
```

## Import

```tsx
import { Tabs } from '@ui/components';
```

## Usage

```tsx
export default function TabsDemo() {
  return (
    <Tabs
      tabs={[
        {
          value: 'account',
          label: 'Account',
          content: <AccountSettings />,
        },
        {
          value: 'notifications',
          label: 'Notifications',
          content: <NotificationSettings />,
        },
      ]}
    />
  );
}
```

## Features

- Flexible tab configuration with values, labels and content
- Controlled and uncontrolled modes
- Customizable styling via className props
- Accessible keyboard navigation
- Mobile responsive

## Props

### Root

| Prop                 | Type                      | Description                         |
| -------------------- | ------------------------- | ----------------------------------- |
| `tabs`               | `TabItem[]`               | Array of tab items to display       |
| `value`              | `string`                  | Currently selected tab value        |
| `onValueChange`      | `(value: string) => void` | Callback when tab selection changes |
| `containerClassName` | `string`                  | Class name for the tabs container   |
| `triggerClassName`   | `string`                  | Class name for tab triggers         |
| `listClassName`      | `string`                  | Class name for the tabs list        |
| `contentClassName`   | `string`                  | Class name for tab content          |

### TabItem

```tsx
type TabItem = {
  value: string;
  label: string;
  content?: ReactNode;
};
```

## Examples

### Controlled Tabs

```tsx
import { Tabs } from '@ui/components';

export function ControlledTabs() {
  const [tab, setTab] = useState('account');

  return (
    <Tabs
      value={tab}
      onValueChange={setTab}
      tabs={[
        {
          value: 'account',
          label: 'Account',
          content: <AccountTab />,
        },
        {
          value: 'password',
          label: 'Password',
          content: <PasswordTab />,
        },
      ]}
    />
  );
}
```

### Custom Styling

```tsx
<Tabs
  containerClassName="gap-6"
  listClassName="bg-muted rounded-xl p-2"
  triggerClassName="px-4 py-2 rounded-lg"
  contentClassName="mt-4 p-4"
  tabs={[
    {
      value: 'tab1',
      label: 'Tab 1',
      content: <div>Tab 1 content</div>,
    },
    {
      value: 'tab2',
      label: 'Tab 2',
      content: <div>Tab 2 content</div>,
    },
  ]}
/>
```
