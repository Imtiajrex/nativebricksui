# TimePicker

A component for selecting time with hours, minutes, and optional seconds.

## Installation

```bash
npm install @ui/components
```

## Import

```tsx
import { TimePicker } from '@ui/components';
```

## Usage

```tsx
export default function TimePickerDemo() {
  const [time, setTime] = useState<string>();

  return <TimePicker value={time} onChange={setTime} placeholder="Select time" />;
}
```

## Features

- Hour and minute selection
- Optional seconds selection
- 12/24 hour format
- Input field with time picker popover
- Keyboard navigation
- Form integration
- Customizable styling

## Props

### Root

| Prop                 | Type                     | Description                     |
| -------------------- | ------------------------ | ------------------------------- |
| `value`              | `string`                 | Selected time value             |
| `onChange`           | `(time: string) => void` | Time selection callback         |
| `placeholder`        | `string`                 | Input placeholder text          |
| `format`             | `'12' \| '24'`           | Time format (default: '24')     |
| `pickSecond`         | `boolean`                | Enable seconds selection        |
| `disabled`           | `boolean`                | Disable the input               |
| `error`              | `boolean`                | Show error state                |
| `className`          | `string`                 | Optional class name for styling |
| `containerClassName` | `string`                 | Class name for container        |
| `timePickerProps`    | `TimePickerProps`        | Props for time picker component |

## Examples

### Basic Usage

```tsx
<TimePicker value={time} onChange={setTime} placeholder="Select time" />
```

### With Seconds

```tsx
<TimePicker value={time} onChange={setTime} pickSecond placeholder="Select time with seconds" />
```

### 12-Hour Format

```tsx
<TimePicker value={time} onChange={setTime} format="12" placeholder="Select time (12-hour)" />
```

### Form Integration

```tsx
<form onSubmit={handleSubmit}>
  <TimePicker
    value={time}
    onChange={setTime}
    name="appointmentTime"
    required
    placeholder="Appointment time"
  />
  <Button type="submit">Schedule</Button>
</form>
```

### Custom Styling

```tsx
<TimePicker
  value={time}
  onChange={setTime}
  placeholder="Custom styled picker"
  containerClassName="w-full max-w-xs"
  className="rounded-xl border-2"
  timePickerProps={{
    containerClassName: 'p-4',
    className: 'gap-4',
  }}
/>
```

### With Validation

```tsx
<TimePicker
  value={time}
  onChange={setTime}
  error={!time}
  required
  placeholder="Required time"
  message={!time ? 'Please select a time' : undefined}
/>
```
