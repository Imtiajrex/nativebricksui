# ActionSheet

A modal dialog that slides up from the bottom of the screen, typically used on mobile devices to display a list of actions.

## Installation

```bash
npm install @ui/components
```

## Import

```tsx
import { ActionSheet } from '@ui/components';
```

## Usage

```tsx
export default function ActionSheetDemo() {
  const [open, setOpen] = useState(false);

  return (
    <>
      <Button onClick={() => setOpen(true)}>Open ActionSheet</Button>
      <ActionSheet
        open={open}
        onOpenChange={setOpen}
        items={[
          {
            label: 'Edit',
            icon: 'edit',
            onPress: () => console.log('Edit pressed'),
          },
          {
            label: 'Share',
            icon: 'share',
            onPress: () => console.log('Share pressed'),
          },
          {
            label: 'Delete',
            icon: 'trash',
            onPress: () => console.log('Delete pressed'),
            variant: 'destructive',
          },
        ]}
      />
    </>
  );
}
```

## Features

- Bottom sheet interaction pattern
- Customizable action items with icons
- Support for destructive actions
- Smooth animations
- Backdrop overlay with tap to dismiss
- Native-like feel on mobile devices

## Props

### Root

| Prop | Type | Description |
| ---------------------- | --------------------- | ------------------------------- |
| `open` | `boolean` | Whether the action sheet is open |
| `onOpenChange` | `(open: boolean) => void` | Callback when open state changes |
| `items` | `ActionSheetItem[]` | Array of action items to display |
| `title` | `string` | Optional title for the action sheet |
| `description` | `string` | Optional description text |
| `className` | `string` | Optional class name for styling |

### ActionSheetItem

```tsx
type ActionSheetItem = {
  label: string;
  icon?: string;
  onPress: () => void;
  variant?: 'default' | 'destructive';
  disabled?: boolean;
  className?: string;
};
```

## Examples

### Basic ActionSheet

```tsx
<ActionSheet
  open={open}
  onOpenChange={setOpen}
  items={[
    {
      label: 'View Profile',
      icon: 'user',
      onPress: () => handleViewProfile(),
    },
    {
      label: 'Settings',
      icon: 'settings',
      onPress: () => handleSettings(),
    },
  ]}
/>
```

### With Title and Description

```tsx
<ActionSheet
  open={open}
  onOpenChange={setOpen}
  title="Manage Account"
  description="Choose an action to perform on your account"
  items={[
    {
      label: 'Update Profile',
      icon: 'pencil',
      onPress: () => handleUpdate(),
    },
    {
      label: 'Delete Account',
      icon: 'trash',
      onPress: () => handleDelete(),
      variant: 'destructive',
    },
  ]}
/>
```

### With Disabled Items

```tsx
<ActionSheet
  open={open}
  onOpenChange={setOpen}
  items={[
    {
      label: 'Available Action',
      icon: 'check',
      onPress: () => handleAction(),
    },
    {
      label: 'Disabled Action',
      icon: 'lock',
      onPress: () => {},
      disabled: true,
    },
  ]}
/>
```
